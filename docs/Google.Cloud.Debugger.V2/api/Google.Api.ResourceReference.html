ChildType

The resource type of a child collection that the annotated field references. This is useful for parent fields where a resource has more than one possible type of parent.

Example:

message ListLogEntriesRequest { string parent = 1 [(google.api.resource_reference) = { child_type: "logging.googleapis.com/LogEntry" }; }
