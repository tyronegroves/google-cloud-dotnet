Filter

Required.

The filter string specifies the jobs to be enumerated.

Supported operator: =, AND

The fields eligible for filtering are:

  • companyName (Required)
  • requisitionId (Optional)
  • status (Optional) Available values: OPEN, EXPIRED, ALL. Defaults to OPEN if no value is specified.

Sample Query:

  • companyName = "projects/api-test-project/companies/123"
  • companyName = "projects/api-test-project/companies/123" AND requisitionId = "req-1"
  • companyName = "projects/api-test-project/companies/123" AND status = "EXPIRED"

Optional.

Default is 100 if empty or a number < 1 is specified.
